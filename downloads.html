    Client & Server Packages

    FoundationDB is available for download on the following platforms and the latest releases are available on the GitHub releases page.

    macOS

    The macOS installation package is supported on macOS 10.7+. It includes the client and (optionally) the server.

    • FoundationDB-7.3.63_x86_64.pkg

    • FoundationDB-7.3.63_arm64.pkg

    Ubuntu

    The Ubuntu packages are supported on 64-bit Ubuntu 12.04+, but beware of the Linux kernel bug in Ubuntu 12.x.

    • foundationdb-clients_7.3.63-1_amd64.deb

    • foundationdb-server_7.3.63-1_amd64.deb (depends on the clients package)

    RHEL/CentOS 7

    The RHEL/CentOS EL7 packages are supported on 64-bit RHEL/CentOS 7.x.

    • foundationdb-clients-7.3.63-1.el7.x86_64.rpm

    • foundationdb-server-7.3.63-1.el7.x86_64.rpm (depends on the clients package)

    Windows

    The Windows installer is supported on 64-bit Windows XP and later. It includes the client and (optionally) the server (for FoundationDB versions 6.3.12 and older).

    • foundationdb-6.3.12-x64.msi
